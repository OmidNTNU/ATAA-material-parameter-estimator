Constitutive Parameter Estimator for Ascending Thoracic Aortic Aneurysms (ATAA)

An interactive tool to find the closest material parameters from a clinical dataset.

How to Use This Tool

This tool uses a k-Nearest Neighbors machine learning model to find the most similar patient dataset based on the clinical parameters you provide. Please follow these steps:

  1. Enter the clinical information for your patient in the form below.
  2. Click the "Find Parameters" button.
  3. The model will process the inputs and display the constitutive parameters from the closest matching patient in our database.

Strain Energy Function (SEF)

The retrieved parameters are intended for use in a Holzapfel-Gasser-Ogden (HGO) type anisotropic hyperelastic model. The strain energy function is given by:

$$ \Psi = \frac{\mu}{2}(\bar{I}_1 - 3) + \frac{k_1}{2k_2} \sum_{i=4,6} \left( e^{k_2[\kappa \bar{I}_1 + (1-3\kappa)\bar{I}_i - 1]^2} - 1 \right) $$

Where $\bar{I}_1$ and $\bar{I}_{4,6}$ are invariants of the right Cauchy-Green tensor, and the parameters $\mu, k_1, k_2, \kappa$ (dispersion), and fiber angles determine the material response.

Note: The simplified model used for parameter fitting may use a slightly different formulation. Please refer to the source paper for the exact form. The parameters `k_op`, `k_ip1`, `k_ip2`, `alpha1`, `alpha2` correspond to the specific implementation.
